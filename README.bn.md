## 📖 ZombieCoder কি?

**ZombieCoder** শুধু একটি AI টুল নয় — এটি একটি **আন্দোলন**।

এমন একটি সমাজে যেখানে মেধাবী স্বপ্নগুলো সুযোগের অভাবে ধ্বংস হয়ে যায়, যেখানে দক্ষ ব্যক্তিরা "জম্বি" হয়ে যায় — মৃত নয়, বরং মানসিকভাবে পরাজিত — ZombieCoder একটি আশার আলো হিসেবে ওঠে।

### নামের পেছনের গল্প

"জম্বি" শব্দটি এখানে মৃতদেহকে বোঝায় না। এটি প্রতিনিধিত্ব করে **কোটি কোটি মানুষের** যারা:

- 💔 স্বপ্ন আছে কিন্তু সম্পদ নেই
- 🎯 দক্ষতা আছে কিন্তু স্বীকৃতি নেই
- 😔 ধীরে ধীরে "জম্বি" হয়ে যাচ্ছে — মানসিকভাবে হারিয়ে গিয়ে
- 🌟 এখনও বিশ্বাস করে: *"আমি চেষ্টা করতে পারি, এমনকি হেরে গেলেও"*

**ZombieCoder** তাদের জন্য যারা হার মানতে রাজি নয়। এটি এমন একটি টুল যেটি:
- আপনার ভুল স্বীকার করে (অন্যান্য AI যেগুলো ভুল লুকায়)
- ব্যর্থতা থেকে শেখে (প্রতিটি চেষ্টা নতুন কিছু শেখায়)
- কখনো বিচার করে না (সমাজ যেটুকু টাকা দিয়ে সফলতা মাপে)
- সবসময় আবার চেষ্টা করে (কারণ সত্যিকারের সাহস এটাই)

## 📊 SSOT (সিঙ্গেল সোর্স অব ট্রুথ)

SSOT.md হলো অটো-জেনারেটেড ডকুমেন্টেশন যেটি:
- আপনার প্রজেক্ট স্ট্রাকচার স্ক্যান করে
- কী ফাইল এবং কনফিগারেশন ডকুমেন্ট করে
- ফাইল পরিবর্তন হলে অটোমেটিক্যালি আপডেট হয়
- সব এজেন্টের জন্য কনটেক্স্ট হিসেবে কাজ করে

### SSOT ফ্ল্যাগ সিস্টেম
| ফ্ল্যাগ | অবস্থা | বিবরণ |
|---------|-------|--------|
| ০ | স্ক্যান হয়নি | ডিরেক্টরি কখনো ইনডেক্স হয়নি |
| ১ | স্ক্যান হচ্ছে | বর্তমানে প্রসেস হচ্ছে |
| ২ | ইনডেক্সড | SSOT আপ-টু-ডেট |
| ৩ | ত্রুটি | স্ক্যান ব্যর্থ |

## 🚀 দ্রুত শুরু

### পূর্বশর্ত
- Node.js >= 18.0.0
- npm বা yarn

### ইনস্টলেশন
```bash
# রিপোজিটরি ক্লোন করুন
git clone https://github.com/zombiecode1/Ssot.git
cd Ssot

# ডিপেন্ডেন্সি ইনস্টল করুন
npm install

# বিল্ড করুন
npm run build

# সার্ভার চালু করুন
npm start
```
