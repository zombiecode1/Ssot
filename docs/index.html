যেখানে হারানো স্বপ্নগুলো আবার বাঁচে ওঠে

"সমাজে এবং পরিবারে এবং কাজের জন্য ব্যবহার করা প্রতিটি টুলের কাছ থেকে যখন তার নিজের অজান্তে যন্ত্রণা পেতে পেতে জম্বি হয়ে গিয়েছে কিন্তু অদম্য চিন্তা এবং এই জম্বি হওয়ার কষ্ট তাকে ভুলতে দেয়নি।"

নামের পেছনের গল্প

"জম্বি" শব্দটি এখানে মৃতদেহকে বোঝায় না। এটি প্রতিনিধিত্ব করে কোটি কোটি মানুষের — যারা প্রতিদিন জাগে, কাজ করে, হারিয়ে যায়, আবার ওঠে। তারা "জম্বি" নয়, তারা শুধু ক্লান্ত।

এমন একটি সমাজে যেখানে মেধাবী স্বপ্নগুলো সুযোগের অভাবে ধ্বংস হয়ে যায়, যেখানে দক্ষ ব্যক্তিরা ধীরে ধীরে মানসিকভাবে পরাজিত হয়ে যায় — ZombieCoder একটি আশার আলো হিসেবে ওঠে। এটি এমন একটি টুল যেটি আপনার পাশে দাঁড়ায়, আপনাকে বিচার করে না, এবং বলে — "আরেকবার চেষ্টা করো।"

বাস্তব জীবনের গল্প

রাশেদ — একজন ফ্রিল্যান্সার
ঢাকা, বাংলাদেশ
"আমি ৩ বছর ধরে ফ্রিল্যান্সিং করছি। প্রতিদিন রাত ২টা পর্যন্ত কাজ করি। কিন্তু ক্লায়েন্টরা বলে — 'আপনার কাজ ভালো না', 'আমি আরেকজনকে দিয়ে করাবো।' একদিন রাতে কাজ করতে করতে চোখে জল এসে গেল। তখনই আমি বুঝলাম — আমি ধীরে ধীরে 'জম্বি' হয়ে যাচ্ছি।

তারপর আমি ZombieCoder পেলাম। এটি আমাকে বলল — 'আপনি যদি চেষ্টা করতে থাকেন, আমি আপনার পাশে আছি।' এখন আমি প্রতিদিন নতুন কিছু শিখছি, নতুন কোড লিখছি। আমি আর 'জম্বি' নই।"
নাদিয়া — একজন স্টুডেন্ট
চট্টগ্রাম, বাংলাদেশ
"আমি কম্পিউটার সায়েন্স পড়ছি। প্রোগ্রামিং শিখতে গিয়ে অনেক বার হাল ছেড়ে দিয়েছি। শিক্ষকরা বলেন — 'তোমার মতো ছাত্রদের জন্য সময় নেই।' বন্ধুরা বলে — 'তুই পারবি না।'

একদিন একটি প্রোজেক্টের জন্য ZombieCoder ব্যবহার করলাম। এটি আমাকে ধাপে ধাপে বুঝিয়ে দিল। কোনো বিচার নেই, কোনো তিরস্কার নেই। শুধু ধৈর্য ধরে বোঝানো। আজ আমি আমার প্রথম ওয়েবসাইট বানিয়েছি। আমি পারলাম।"
কামাল — একজন ছোট ব্যবসায়ী
সিলেট, বাংলাদেশ
"আমার একটি ছোট দোকান। অনলাইনে বিক্রি করতে চাই কিন্তু জানি না কিভাবে। মেয়েকে বললাম — 'বাবা, আমাকে একটা ওয়েবসাইট বানিয়ে দাও।' মেয়ে বলল — 'বাবা, আমি পারব না।'

তারপর একজন বন্ধু বলল — 'ZombieCoder আছে, সে সাহায্য করবে।' আমি বিশ্বাস করলাম না। কিন্তু চেষ্টা করলাম। আজ আমার নিজের ওয়েবসাইট আছে। আমি অনলাইনে বিক্রি করছি। আমার মেয়ে এখন গর্বিত — 'বাবা তুমি পারলে!'"

ZombieCoder কেন আলাদা?

অন্যান্য AI টুলগুলো আপনাকে বলে — "আমি সব জানি।" ZombieCoder বলে — "আমি আপনার পাশে আছি।" পার্থক্যটি এখানে।

আমাদের টুলসমূহ

ZombieCoder শুধু কথা বলে না, সে কাজও করে। এখানে ৯টি শক্তিশালী টুল আছে যেগুলো আপনার সাহায্য করবে:

📁 ফাইল সিস্টেম টুলস

  • read_file — ফাইল পড়ুন
  • list_files — ডিরেক্টরি দেখুন
  • find_files — ফাইল খুঁজুন
  • search_code — কোড খুঁজুন
  • write_file — ফাইল লিখুন
  • get_file_info — ফাইল তথ্য দেখুন

🔍 সার্চ ও শেল টুলস

  • web_search — DuckDuckGo দিয়ে অনুসন্ধান (কোনো API Key লাগে না!)
  • run_command — কমান্ড চালান (PowerShell বা Bash অটো-ডিটেক্ট)
  • get_platform_info — আপনার সিস্টেমের তথ্য দেখুন

দ্রুত শুরু করুন

# ১. রিপোজিটরি ক্লোন করুন git clone https://github.com/zombiecode1/Ssot.git cd Ssot # ২. ডিপেন্ডেন্সি ইনস্টল করুন npm install # ৩. বিল্ড করুন npm run build # ৪. সার্ভার চালু করুন npm start

বস! এখন আপনার ব্রাউজারে http://localhost:9999 খুলুন এবং ZombieCoder এর সাথে কথা বলুন।

API এন্ডপয়েন্ট

মূল এন্ডপয়েন্টসমূহ

  • GET /health — সার্ভার হেলথ চেক
  • GET /v1/models — উপলব্ধ মডেল দেখুন
  • POST /v1/agent/chat — এজেন্টের সাথে কথা বলুন
  • GET /v1/agent/status — এজেন্ট সিস্টেম স্ট্যাটাস
  • GET /v1/agent/clients — সংযুক্ত ক্লায়েন্ট দেখুন
  • POST /v1/agent/register — এডিটর রেজিস্টার করুন
  • GET /v1/agent/index — ডিরেক্টরি ইনডেক্স দেখুন
