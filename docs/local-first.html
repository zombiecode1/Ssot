Local First — সত্যি কথা

"লোকাল" শব্দটি দিয়ে আমরা কী বুঝি, এবং বাস্তবে কী ঘটে — এই দুটোর মধ্যে পার্থক্য আছে। আজ সেই পার্থক্য স্বীকার করি।

❌ "ডাটা বাহিরে যায় না" — এটি মিথ্যা

সত্যি কথা:

আপনার ডাটা বাহিরে যায়। প্রতিটি API কল, প্রতিটি নেটওয়ার্ক রিকোয়েস্ট, প্রতিটি সফটওয়্যার আপডেট — সবকিছু ডাটা বাইরে পাঠায়। এটি স্বীকার না করা অসততা।

আমরা যখন বলি "ZombieCoder লোকাল-ফার্স্ট", তখন আমরা একটি আশ্বাস দিই যেটি বাস্তবতার সাথে মানানসই নয়। সত্যি কথা হলো — আপনার কম্পিউটারে কিছুই সম্পূর্ণ লোকাল নয়।

🔍 বাস্তবে কী "লোকাল"?

"লোকাল" বলতে আমরা কী বুঝেছিলাম? — যেটুকু আপনার নিয়ন্ত্রণে আছে, যেটুকু আপনার কম্পিউটারে চলছে, যেটুকু দ্রুত কাজ করছে। কিন্তু এটি কি সত্যিই "লোকাল"?

"বাস্তবিক অর্থে লোকাল বলতে দুনিয়ায় কিছুই নেই। কেন বলছি? আপনার কম্পিউটারের মাদারবোর্ড Intel, গ্রাফিক্স কার্ড NVIDIA, অপারেটিং সিস্টেম Windows, এডিটর VS Code, GitHub, ব্রাউজার Google Chrome — এগুলো ব্যতীত একজন মানুষ সভ্য সমাজে এখন থাকতে পারে না।"

— সাহন স্রাবন, ZombieCoder এর প্রতিষ্ঠাতা

🌐 আপনার "লোকাল" কম্পিউটারের বাইরের নির্ভরশীলতা

  • 🔧
    মাদারবোর্ড — Intel / AMD
    চীনা, তাইওয়ানি, আমেরিকান কারখানায় তৈরি। প্রতিটি চিপে মাইক্রোকোড আছে যেটা নিয়ন্ত্রণ করে Intel।
  • 🎮
    গ্রাফিক্স কার্ড — NVIDIA / AMD
    ড্রাইভার ডাউনলোড করতে NVIDIA সার্ভারে যেতে হয়। প্রতিটি আপডেট ডাটা পাঠায়।
  • 💻
    অপারেটিং সিস্টেম — Windows
    Microsoft এর সার্ভার থেকে ডাউনলোড। টেলিমেট্রি পাঠায়। আপডেটের জন্য ইন্টারনেট লাগে।
  • 📝
    এডিটর — VS Code / Cursor
    GitHub (Microsoft) এর সার্ভার থেকে ডাউনলোড। এক্সটেনশন মার্কেটপ্লেস থেকে আসে।
  • 🔀
    Git & GitHub
    কোড স্টোর হয় Microsoft এর সার্ভারে। প্রতিটি push ডাটা বাইরে পাঠায়।
  • 🌐
    ব্রাউজার — Google Chrome
    Google Chromium এর উপর ভিত্তি। V8 ইঞ্জিন Google কন্ট্রোল করে। টেলিমেট্রি Google এ যায়।
  • 📱
    অ্যান্ড্রয়েড — Google
    ফোনের প্রতিটি অ্যাপ Google Play Store এর মাধ্যমে আসে। Google সবকিছু ট্র্যাক করে।

বাস্তবতা:

এগুলো ছাড়া আপনি সভ্য সমাজে থাকতে পারবেন না। তাহলে "লোকাল" বলতে কী বোঝা যায়?

✅ তাহলে "লোকাল-ফার্স্ট" দিয়ে আমরা কী বোঝাই?

আমরা যখন "লোকাল-ফার্স্ট" বলি, তখন আমরা বোঝাই — আপনার নিয়ন্ত্রণের সীমার মধ্যে সর্বোচ্চ ক্ষমতা আপনার হাতে দেওয়া। এটি সম্পূর্ণ লোকাল নয়, কিন্তু এটি আপনার পক্ষে সেরা সম্ভাব্য বিকল্প।

বিষয় সম্পূর্ণ বাইরের (Cloud) ZombieCoder (লোকাল-ফার্স্ট)
AI মডেল OpenAI/Google সার্ভারে চলে আপনার কম্পিউটারে চলে (লোকাল LLM)
ডাটা স্টোরেজ Cloud সার্ভারে থাকে আপনার হার্ড ডিস্কে থাকে
কনভারসেশন কোম্পানির ডাটাবেসে সেভ হয় SQLite ফাইলে আপনার কম্পিউটারে সেভ হয়
নেটওয়ার্ক ইন্টারনেট ছাড়া কাজ করে না ইন্টারনেট ছাড়াও কাজ করে (লোকাল মডেল)
কোস্ট মাসিক সাবস্ক্রিপশন লাগে সম্পূর্ণ বিনামূল্যে

🤝 আমরা কী প্রতিশ্রুতি দিই?

আমাদের সত্যিকারের প্রতিশ্রুতি:

  • ✅ AI মডেলটি আপনার কম্পিউটারেই চলবে (লোকাল LLM)
  • ✅ আপনার কনভারসেশন ডাটা আপনার কম্পিউটারেই সেভ হবে
  • ✅ আমরা আপনার ডাটা তৃতীয় পক্ষকে বিক্রি করব না
  • ✅ আপনি যখন চান, তখন সবকিছু ডিলিট করতে পারবেন
  • ✅ সোর্স কোড Open Source — আপনি যাচাই করতে পারবেন

আমরা যা স্বীকার করি:

  • ❌ সম্পূর্ণ লোকাল — এটি সম্ভব নয়
  • ❌ কোনো ডাটা বাইরে যায় না — এটি মিথ্যা
  • ❌ ইন্টারনেট ছাড়া সবকিছু কাজ করে — এটি অসম্ভব
  • ❌ প্রাইভেসি 100% — এটি বাস্তবসম্মত নয়

💡 আমাদের দর্শন

"আমরা বিশ্বাস করি — আপনি আপনার সিস্টেমের মালিক। আমরা শুধু একটি টুল দিই যেটি আপনার সাথে মিথ্যা বলে না। আমরা বলি — 'এটি আপনার কম্পিউটারে চলে, এটি আপনার ডাটা রাখে, এবং এটি আপনার নিয়ন্ত্রণে।' এটি সত্যি কথা। এটিই আমাদের প্রতিশ্রুতি।"

— ZombieCoder টিম

"লোকাল-ফার্স্ট" শব্দটি দিয়ে আমরা বোঝাই — আপনি আপনার ডাটার উপর সর্বোচ্চ নিয়ন্ত্রণ রাখতে পারেন। আপনি জানবেন আপনার ডাটা কোথায় আছে, কে দেখতে পারে, এবং আপনি কখন ডিলিট করতে চান। এটি সম্পূর্ণ প্রাইভেসি নয়, কিন্তু এটি সম্ভাব্য সেরা প্রাইভেসি।

🎯 উপসংহার

আমরা মিথ্যা বলি না। আমরা সত্যি কথা বলি। "লোকাল-ফার্স্ট" মানে নয় যে কিছুই বাইরে যায় না। "লোকাল-ফার্স্ট" মানে — আপনি আপনার সিস্টেমের মালিক, এবং আমরা সেই মালিকানাকে সম্মান করি।

প্রতিটি সফটওয়্যার, প্রতিটি হার্ডওয়্যার, প্রতিটি অপারেটিং সিস্টেম — সবকিছু বাইরের নির্ভরশীলতা রাখে। কিন্তু আমরা সেই নির্ভরশীলতাকে কমিয়ে আনতে চাই। আমরা চাই — আপনি কম নির্ভরশীল হোন, বেশি স্বাধীন হোন।
