## рҹҡҖ Quick Start

### Prerequisites
- Node.js >= 18.0.0
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/zombiecode1/Ssot.git
cd Ssot

# Install dependencies
npm install

# Build
npm run build

# Start server
npm start
```

That's it! Now open `http://localhost:9999` and talk to ZombieCoder.

## рҹ“Ҡ SSOT (Single Source of Truth)

SSOT.md is auto-generated documentation that:

- вң… Scans your project structure
- вң… Documents key files and configurations
- вң… Auto-updates when files change
- вң… Provides context for all agents

### SSOT Flag System

| Flag | Status | Description |
|------|--------|-------------|
| 0 | Not scanned | Directory never indexed |
| 1 | Scanning | Currently processing |
| 2 | Indexed | SSOT up-to-date |
| 3 | Error | Scan failed |
